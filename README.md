# Meme generator

**Meme generator** is a web application for creating memes, only need from you is think about text image will be randomly provided. Build using React, Javascript and Vite


### Prerequisites

Before you begin, ensure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (version 14.x or later)
- [npm](https://www.npmjs.com/) (comes with Node.js) or [Yarn](https://yarnpkg.com/)

### Installation

1. **Clone the Repository**

   ```bash
   git clone https://github.com/ProbablyNoth1ng/meme-generator.git
   ```   
2. **Navigate to the Project Directory**

   ```bash
   cd meme-generator
   ```

4. **Install Dependencies**
    
    ```bash
   npm install 
    ```
    or
   ```bash
    yarn install
    ```

6. **Running the Project**
    
    ```bash
    npm run dev
   ```
    or
   ```bash
     yarn dev
   ```

8. **Building for Production**
    
    ```bash
    npm run build 
   ```
    or
   ```bash
   yarn build
   ```
